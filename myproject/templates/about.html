Iris Flower Classification

One of the very famous classification problems in Machine Learning is the

IRIS Flower classification problem.Iris Flower has different species :

(Setosa, Versicolor, Verginica).



Problem Statement

Given four attribute values(Sepal Length,Sepal Width,Petal Length,Petal Width)

the class of Iris(Setosa, Versicolor, Verginica).


Data Description

Data Attributes :

  1. Sepal Length
  2. Sepal Width
  3. Petal Length
  4. Petal Width
  5. Class
    • Setosaa
    • Versicolor
    • Verginica
